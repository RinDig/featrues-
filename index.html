Core Platform Features
Foundation features that multiply the value of every test. These create immediate impact while building infrastructure for scale.
Time-Series Testing
Quick Win High ROI
What It Does
Run the same test at different time intervals (daily, weekly, monthly) to track how perception shifts. Automatically re-engage past participants or fresh audiences. Shows trend lines of how associations with the same images evolve over time, especially useful during campaigns or cultural moments.
Business Value
Perception isn't static. A political message that resonates today might fail tomorrow after news events. Gaming companies can track hype decay/growth leading to launch. Universities can see how perception shifts through application season. This longitudinal data justifies ongoing retainers ($10K/month) rather than one-time projects.
Implementation
Scheduling system for recurring tests. Trend visualization dashboard. Automated participant re-engagement. Change detection alerts. Development time: 2 weeks.
Demographic Filtering
Enterprise
What It Does
Real-time filtering of heat map data by age, gender, location, income, and custom segments. Clients can instantly see how different demographics respond to the same visual, with results updating live as filters change.
Business Value
This is table stakes for enterprise clients. J.Crew needs to know if Gen Z women respond differently than millennial men. Political campaigns must see swing voter responses separate from base supporters. Without this, you're locked out of contracts over $50K. With it, you can charge 2-3x more because insights become actionable by audience.
Implementation
Add demographic fields to participant data collection. Create indexed database queries with filter parameters. Front-end filter UI with state management. Development time: 2-3 weeks.
Mobile-First Testing
Quick Win
What It Does
Optimized interface for smartphone participation. Touch-friendly heat mapping, swipe gestures for navigation, vertical layouts, and reduced data usage. Works perfectly on any device from iPhone SE to iPad Pro.
Business Value
70% of consumer audiences are on mobile. Without mobile optimization, you're excluding most participants or getting poor-quality data from frustrated users. College students exclusively use phones. Gaming audiences test during commutes. This feature directly impacts response rates and data quality, which clients notice immediately.
Implementation
Responsive CSS refactor. Touch event handlers. Progressive image loading. Viewport optimizations. Development time: 1-2 weeks. Mostly front-end work.

AI-Powered Features
Intelligent automation that turns data into insights. These features reduce manual work while uncovering patterns humans miss.
Auto-Tag Generator
Quick Win High ROI
What It Does
When clients upload images, AI automatically analyzes and tags them with style attributes (minimal, bold, vintage), color schemes, emotions (energetic, calm), and industry categories. Tags become searchable and filterable across all tests.
Business Value
Clients spend hours categorizing images before testing. Auto-tagging saves 2-3 hours per project and enables powerful cross-test insights like "vintage styles consistently outperform modern across all our campaigns." For agencies running 10 tests monthly, this saves 30 hours of billable time they can redirect to strategy. It also builds a taxonomy database over time that becomes a competitive moat.
Implementation
Single API call to GPT-4 Vision or Claude. Pass image, get structured JSON tags back. Store in database. Build simple UI for tag management. Development time: 3-4 days.
Report Writer
High ROI
What It Does
Automatically generates executive summaries, insight reports, and recommendations from heat map data. Customizable tone (formal for boards, casual for creatives) and length. Includes key findings, statistical significance, and next steps.
Business Value
Agencies charge $2-5K just for insight reports. This feature generates them in minutes, not days. More importantly, it helps non-analytical clients understand what the data means. A CMO doesn't want heat maps - they want to know "Young consumers prefer our heritage designs 3:1 over modern concepts." This turns data into decisions and justifies premium pricing.
Implementation
Create report templates with variable slots. Pass heat map data + metadata to GPT-4. Generate sections iteratively. Add client branding. Export as PDF. Development time: 1-2 weeks.
Anomaly Detection
Enterprise
What It Does
Identifies unusual response patterns that might indicate bot activity, sampling bias, or data quality issues. Flags when results deviate significantly from expected distributions. Can detect coordinated responses, speed-running, and random clicking.
Business Value
Data quality is everything. One bad dataset can tank a campaign worth millions. Political campaigns are especially vulnerable to coordinated manipulation. This feature protects the integrity of insights, which protects client relationships. It also enables "verified data" premium pricing - charge 20% more for quality-assured results.
Implementation
Start with statistical analysis (standard deviation, time-based patterns). Add ML model later for sophisticated detection. Flag suspicious responses for review. Development time: 2 weeks for basic, 2 months for ML.

Workflow Features
Streamline operations from upload to insight. These features reduce friction and accelerate time-to-value for every test.
Template Library
Quick Win High ROI
What It Does
Pre-configured test templates for common use cases: movie posters (4 variations), logo testing (6 options), campaign visuals (hero + supporting), university recruitment (lifestyle + academic), etc. One-click setup with smart defaults for audience size, demographics, and questions.
Business Value
Reduces new user setup time from 30 minutes to 2 minutes. More importantly, it shows clients what's possible. A gaming studio might not think to test character designs until they see the template. Templates become upsell opportunities - start with poster testing, expand to full campaign. They also standardize output, making cross-client benchmarking possible.
Implementation
JSON configuration files defining test structure. Simple template selector in UI. Ability to save custom templates. Development time: 1 week for 10 templates.
Batch Upload
Quick Win
What It Does
Drag and drop 50+ images at once into a single test or across multiple test groups. Automatic organization by naming patterns. Bulk operations for tagging, categorizing, and screen assignment. Smart grouping for variation sets (poster_v1, poster_v2, etc).
Business Value
Agencies often test entire campaigns (30-50 assets). Without batch upload, they spend an hour just uploading files one by one. This feature turns Constellations from a "single test" tool to a "campaign testing platform." It enables bigger deals - instead of selling 5 tests at $7.5K each, sell a 50-asset campaign test for $75K.
Implementation
Modern file upload library (Dropzone.js or similar). Backend queue processing. Progress WebSocket updates. Folder organization logic. Development time: 1 week.
White-Label Testing Links
Enterprise High ROI
What It Does
Clients can send tests from their own domain (test.clientbrand.com) with their branding. Custom colors, logos, and messaging throughout the testing experience. Participants never see Constellations branding.
Business Value
This single feature can double your enterprise pricing. Big brands won't send customers to a third-party domain - it breaks trust and looks unprofessional. White-labeling transforms you from a vendor to invisible infrastructure. Universities can test from admissions.college.edu. Political campaigns from candidate.com/feedback. This feature alone justifies $100K+ annual contracts.
Implementation
Subdomain wildcards. Database schema for client branding. CSS theming system. Custom domain CNAME setup. Development time: 2-3 weeks for full implementation.

Analysis Features
Transform raw data into competitive intelligence. These features create unique insights that become your clients' strategic advantage.
Export to Figma/Adobe
Quick Win High ROI
What It Does
One-click export of winning designs, heat maps, and insights directly into Figma or Adobe Creative Cloud. Includes annotations, scores, and demographic breakdowns as layers. Designers stay in their tools while accessing data.
Business Value
Designers hate context switching. This feature keeps them in their creative flow while incorporating data. More importantly, it makes Constellations part of the standard design workflow, not an extra step. Once embedded in daily tools, clients never leave. Agencies using this feature have 3x higher retention rates. It also enables design tool partnerships and co-marketing opportunities.
Implementation
Figma and Adobe APIs are well-documented. Build plugins for each platform. Export format handlers. Authentication flow. Development time: 2 weeks per platform.
Competitive Benchmarking
Enterprise High ROI
What It Does
Compare test results against industry averages built from anonymous, aggregated data. See if your campaign visuals perform above or below category benchmarks. Track trends over time (e.g., "minimal designs up 23% this quarter").
Business Value
This creates a data moat competitors can't match. Every test makes the benchmarks better. Clients pay premium for context - knowing you scored 72% means nothing, knowing you're 15 points above industry average means everything. Gaming companies want to know if their character designs test better than average. Universities want to know if their campaigns resonate more than peer institutions. This feature enables $200K+ annual contracts.
Implementation
Anonymous data aggregation pipeline. Statistical normalization. Industry categorization. Benchmark calculation engine. Privacy controls. Development time: 3-4 weeks.
Custom Scoring Algorithms
Enterprise
What It Does
Clients can weight different demographics, regions, or response types differently in final scores. A luxury brand might weight high-income responses 3x. A university might prioritize out-of-state students. Political campaigns can focus on swing voters.
Business Value
Generic insights are commodity. Custom insights are strategic. This feature transforms Constellations from a testing tool to a strategic platform aligned with specific business goals. It justifies consulting-level pricing because you're not just providing data, you're providing decision intelligence. Clients become dependent on their custom algorithms, reducing churn to near zero.
Implementation
Weighted average calculation engine. UI for setting weights. Algorithm versioning. A/B testing of algorithms. Development time: 2 weeks for basic, 4 weeks for advanced.

Engagement Features
Turn one-time tests into ongoing relationships. These features improve response quality while building valuable audience assets.
Follow-up Surveys
Quick Win High ROI
What It Does
After heat mapping, trigger contextual questions based on choices. If someone dislikes all modern designs, ask why. If they love vintage, explore what era. Questions branch dynamically based on previous answers, creating conversational depth.
Business Value
Heat maps show what. Follow-ups reveal why. This transforms insights from "Young people prefer Design A" to "Young people prefer Design A because it reminds them of streetwear brands they trust." This deeper understanding justifies 2-3x higher pricing. Agencies can charge $15K instead of $5K because they're delivering strategy, not just data. The qualitative insights also make better case studies and client stories.
Implementation
Branching logic engine. Question bank management. Response aggregation. Natural language processing for open-text answers. Development time: 1-2 weeks.
Participant Pools
High ROI
What It Does
Build and maintain your own testing audience. Participants can opt-in to future tests. Tag them by interests, demographics, and past responses. Re-engage for new tests with one click. Track participation history and quality scores.
Business Value
Recruiting participants costs $20-100 per person through panels. Building your own pool eliminates this cost after initial investment. More importantly, it enables longitudinal studies - track how the same audience's preferences evolve. Gaming companies can build pools of their actual players. Universities can maintain pools of prospective students. This creates switching costs - clients won't leave if it means losing their cultivated audience.
Implementation
User registration system. Opt-in management. Email engagement tools. Audience segmentation. Quality scoring algorithm. Development time: 3-4 weeks.
Thank You Personalization
Quick Win
What It Does
Customized completion messages based on client brand and participant behavior. Can include discount codes, early access offers, or insider information. Messages can vary by response patterns - someone who loved everything gets different messaging than a critic.
Business Value
This simple feature can increase response rates by 25-40%. More responses = better data = higher value insights. It also turns testing into a marketing channel. Gaming companies can offer beta access to engaged participants. Retailers can provide exclusive discounts. Universities can invite high-interest students to campus. The feature pays for itself through improved data quality alone, but also enables new revenue streams.
Implementation
Template system with variables. Client-customizable messages. Response-pattern triggers. Integration with client reward systems. Development time: 3-4 days.
